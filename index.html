Sobre mí

Hola, soy Lucas. Durante muchos años me desempeñé como maquinista naval, una etapa que me formó en la disciplina, la responsabilidad y la resolución de problemas en entornos exigentes. Con el tiempo decidí dar un giro profesional hacia el desarrollo de software, un área en la que encontré una nueva forma de construir, mejorar procesos y crear soluciones útiles. Hace dos años trabajo como desarrollador, mientras continúo formándome y fortaleciendo mis conocimientos en tecnologías web. Además, soy una persona que prioriza la familia, un valor que guía mi forma de vivir, de comprometerme con mis objetivos y de proyectar mi crecimiento personal y profesional.

Proyectos y experiencias

Proyecto Interno con .NET y React

Me encuentro trabajando en el desarrollo de un sistema interno utilizando .NET, React, Entity Framework y SQL Server. Dentro de este entorno participo en funcionalidades relacionadas con la firma digital GDE aplicada a fojas de conceptos, colaborando en procesos de desarrollo, integración y mantenimiento. Debido a que se trata de una plataforma interna con información clasificada, no es posible compartir imágenes ni detalles específicos, pero esta experiencia me permite seguir creciendo en backend, frontend, base de datos y mejora de soluciones orientadas a procesos reales.

Sitio Web Corporativo - Sehmat

Desarrollé una landing page profesional para Sehmat, un estudio especializado en seguridad e higiene laboral, con la intención de crear una presencia web clara, moderna y confiable. Trabajé en la organización del contenido para mostrar de forma atractiva sus servicios, credenciales, equipo de trabajo y medios de contacto. Para este proyecto utilicé Git, lo que me permitió llevar un control ordenado de las versiones y actualizar el sitio de forma más segura. Además, la página está publicada en DonWeb, lo que me facilita la conexión con el repositorio y ver reflejadas las actualizaciones de manera rápida en el sitio.

App de Meteorología - METEREOLOGIA LGM

Desarrollé una aplicación web interactiva para consultar el clima en distintas ciudades, mostrando información meteorológica en tiempo real de forma clara y visual. El proyecto fue realizado con HTML, CSS y JavaScript, integrando la API de OpenWeatherMap para obtener datos actualizados. También incorporé una interfaz moderna con fondo dinámico, efectos visuales y diseño responsivo para mejorar la experiencia de usuario. Para el desarrollo utilicé Git y GitHub como control de versiones, organizando los cambios y facilitando la publicación del proyecto.
