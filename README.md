## Descripción del Proyecto
Este proyecto consiste en el desarrollo de una Landing Page de Portafolio Personal utilizando HTML y CSS. El objetivo es presentar mi perfil profesional como desarrollador de software, mostrando mi recorrido desde maquinista naval hasta mi desempeño actual en el área de desarrollo, donde hace dos años trabajo profesionalmente. En el portafolio integré proyectos y experiencias vinculadas al desarrollo interno con .NET, React, Entity Framework y SQL Server, un sitio web corporativo realizado para Sehmat y una aplicación web de meteorología desarrollada con HTML, CSS, JavaScript y consumo de API externa.

## Reflexiones y Decisiones de Diseño

**¿Por qué elegiste esas fuentes?**  
Elegí **Orbitron** para los títulos porque transmite una estética tecnológica y futurista que combina muy bien con la inspiración visual en Star Wars. Para el resto del contenido utilicé **Poppins**, ya que es una fuente moderna, clara y fácil de leer, ideal para mantener buena legibilidad en todo el sitio.

**¿Qué ventajas encontraste al utilizar Grid y Flexbox en tu proyecto?**  
El uso de Grid y Flexbox me permitió organizar de manera clara y adaptable las distintas secciones del portfolio. En especial, la sección de proyectos quedó mejor distribuida y responsiva, logrando que las tarjetas se acomoden correctamente según el tamaño de pantalla sin perder orden ni estética.

**¿Qué animación o transición implementaste y por qué?**  
Implementé efectos hover en las tarjetas, botones e imágenes para dar una sensación más dinámica e interactiva. También apliqué sombras, brillos y transiciones suaves que acompañan la estética general del portfolio, mejorando la presentación visual sin afectar la claridad del contenido.

**¿Qué enfoque personal quise reflejar en este portfolio?**  
Quise mostrar no solo mis conocimientos técnicos, sino también mi recorrido profesional y personal. Mi historia de transición desde maquinista naval hacia el desarrollo de software representa disciplina, adaptación y compromiso con el aprendizaje. Además, busqué transmitir una identidad propia a través de una estética visual alineada con mis gustos personales, especialmente la inspiración en el universo Star Wars.
